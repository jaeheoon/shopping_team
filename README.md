## 쇼핑몰 구현
### (Shopping Mall Team Project)

>- 목차
>1. 프로젝트 목적
>   - 목표
>   - 주제
>   - 구현목표
>2. 유스케이스 모델링
>3. ERD
>4. 테이블정의서
>5. SQL 구조
>6  패키지 구조
>7. 실행화면

## 🛠️ 개발 환경

- **프로그래밍 언어**: Java, jsp
- **IDE**: Eclipse
- **DBMS**: Oracle Database
- **SQL 툴**: SQL Developer
- **버전 관리**: GitHub

## ✨ 프로젝트 목적
### 목표: jsp와 JDBC를 활용해 쇼핑몰 구현
### 주제 : 가방 전문 쇼핑몰 프로젝트 구축
